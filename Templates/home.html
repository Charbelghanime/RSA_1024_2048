Key Generation, Encryption, and Decryption Process

When generating keys, a trusted user has the option to send a message. Here's how the process works:

  1. The RSA key pair (public and private keys) is generated using the selected key size.
  2. A trusted user has the choice to submit a message for encryption.
  3. The generated public key is then utilized to encrypt the message.
  4. The encrypted message is subsequently returned to the user.
  5. The user, holding the private key, can decrypt the message using their private key.
